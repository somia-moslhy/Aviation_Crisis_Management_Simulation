# ✈️ Aviation Crisis Management & Financial Simulation
> **How a 1.5% Failure Rate Impacts the US Economy: A 5.8M Record Deep Dive.**

## 📌 Executive Summary
This project transforms raw operational aviation data (2015 US Flight Dataset) into a **Strategic Financial Simulation**. Instead of just counting delays, we modeled the economic "bleed" caused by cancellations and the unexpected "revenue" generated by diverted flights.

### 🎯 Key Business Insights
* **The Regional Carrier Trap:** Small regional airlines (e.g., American Eagle) suffer disproportionately, losing up to **25% of their estimated revenue** during crises.
* **The Legacy Shield:** Major carriers (e.g., Delta) maintain high resilience, with losses often staying below **2% of revenue** due to massive operational scales.
* **Southwest Anomaly:** Identified as a major financial bleeder in absolute terms, highlighting potential operational stability risks.

---

## 🛠️ Technical Workflow (The 6-Step Pipeline)

1.  **Memory Optimization:** Handled 5.8M rows by implementing strict memory management, dropping redundant features immediately to prevent RAM overflow.
2.  **Defensive Cleaning:** Implemented logical imputation for missing operational data (e.g., mapping cancellation reasons to physical flight realities).
3.  **The October Anomaly Fix:** Resolved an 8.35% data gap caused by FAA system changes using systematic mapping.
4.  **Dynamic Financial Modeling:** Engineered a `Tier-Based Penalty System` using `NumPy Select`.
    * *Premium Airlines:* $75,000 penalty (Brand protection cost).
    * *Budget Airlines:* $25,000 penalty.
    * *Standard:* $50,000 penalty.
5.  **Revenue Estimation:** Simulated base revenue using distance-based proxies ($25/mile) to calculate loss margins.
6.  **Interactive Dashboards:** Built high-impact visuals using **Plotly** to contrast "Financial Bleed" vs. "Operational Gains".
